# Birth & Death Registration Portal

A complete web application for managing birth and death certificate registrations.

## 🚀 Quick Start

### On InfinityFree:
1. Download all files (ZIP)
2. Extract the ZIP file
3. Upload to `htdocs/` folder via File Manager
4. Set folder permissions to 755
5. Visit your domain!
